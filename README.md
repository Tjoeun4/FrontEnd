</p>

# 🎯 프로젝트 개요
**냉가이드**는 1인 가구가 겪는 식재료 관리의 어려움과 식비 부담 등의 **불편함을 해소**하기 위해 기획된 **안드로이드 애플리케이션**입니다.

AI 기반의 **식재료 관리 및 레시피 추천** 기능을 중심으로, **위치 기반 커뮤니티(C2C)**를 통한 식재료 공동구매 및 나눔 기능을 제공합니다. 이를 통해 1인 가구 간의 활발한 상호작용과 합리적인 소비 습관 형성을 돕는 **통합 라이프스타일 서비스**를 지향합니다.

# 🚀 핵심 기능
* **🤖 AI 음식 추천**: 냉장고 속 남은 식재료를 분석하여 최적의 레시피 제안
* **📱 냉장고 관리**: 유통기한 자동 계산 및 알림을 통한 식재료 낭비 방지
* **🤝 C2C 커뮤니티**: 위치 기반 식재료 공동구매 및 이웃 간 무료 나눔
* **💰 스마트 가계부**: 영수증 인식(OCR)을 활용한 편리한 식비 지출 관리
